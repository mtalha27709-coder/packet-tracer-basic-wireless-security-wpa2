# 📡 Packet Tracer - Configure Basic Wireless Security (WPA2)



## 🚀 Overview


This project demonstrates how to configure **basic wireless security using WPA2 Personal** in Cisco Packet Tracer. It simulates a real-world scenario where a small business secures its wireless network from unauthorized access.



---



## 🎯 Objectives


- Configure WPA2 Personal security on a wireless router
- Set up a secure Wi-Fi connection using a passphrase
- Connect a laptop to the secured network
- Verify connectivity after applying security settings



---



## 🧠 Scenario


A small business owner wants to secure their wireless network. Initially, the network has no security enabled. The goal is to implement **WPA2 Personal encryption** to prevent unauthorized access.



---



## ⚙️ Configuration Steps



### 🔹 Part 1: Verify Initial Connectivity


- Open **Laptop → Desktop → Web Browser**
- Enter: `www.cisco.pka`
- Confirm that the webpage loads successfully



---



### 🔹 Part 2: Configure Wireless Security


- Access router via browser: `192.168.1.1`
- Login credentials:
  - Username: `admin`
  - Password: `admin`
- Navigate to:
  - **Wireless → Wireless Security**
 
  - 
- Configure:
  - Set **Security Mode (2.4 GHz)** → `WPA2 Personal`
  - Passphrase → `Network123`
- Save settings



---


### 🔹 Part 3: Connect Laptop to Secure Network


- Go to **Laptop → Desktop → PC Wireless**
- Select network: `Academy`
- Click **Connect**
- Enter password: `Network123`

- 

---



### 🔹 Part 4: Verify Secure Connectivity


- Open Web Browser again
- Enter: `www.cisco.pka`
- Confirm successful access after applying security
